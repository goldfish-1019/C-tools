Mod 1: Star-Delta Transform

Delta to Star (Δ → Y)

Ra = (Rab * Rca) / (Rab + Rbc + Rca)
Rb = (Rbc * Rab) / (Rab + Rbc + Rca)
Rc = (Rca * Rbc) / (Rab + Rbc + Rca)

Mod 2: Network Theorems

Maximum Power Transfer

RL = Rth (For DC)
ZL = Zth* (Complex Conjugate for AC)
Pmax = Vth^2 / (4 * Rth)

Mod 3: Transients & Laplace

Standard Initial Conditions
