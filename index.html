2024 Scheme Exam Cheat Sheet

Mod 1: Star-Delta Transform

Delta to Star (Δ → Y)

Ra = (Rab × Rca) / (Rab + Rbc + Rca)
Rb = (Rbc × Rab) / (Rab + Rbc + Rca)
Rc = (Rca × Rbc) / (Rab + Rbc + Rca)

Star to Delta (Y → Δ)

Rab = (RaRb + RbRc + RcRa) / Rc
Rbc = (RaRb + RbRc + RcRa) / Ra
Rca = (RaRb + RbRc + RcRa) / Rb

Mod 2: Network Theorems

Thevenin's Equivalent

Vth = Open circuit voltage at terminals
Rth = Equivalent resistance (sources dead)

Norton's Equivalent

In = Short circuit current at terminals
Rn = Rth

Maximum Power Transfer

RL = Rth (For DC)
ZL = Zth* (Complex Conjugate for AC)
Pmax = Vth² / (4 × Rth)

Mod 3: Transients & Laplace

Standard Initial Conditions

Inductor: iL(0+) = iL(0-)
Capacitor: vc(0+) = vc(0-)

Laplace Transform Pairs

f(t) F(s)
δ(t) Impulse1
u(t) Step1/s
t (Ramp)1/s²
e-at1/(s+a)
sin(ωt)ω/(s²+ω²)

Mod 4: Two-Port (Z & Y)

Z-Parameters (Open Circuit)

[V1] = [Z11 Z12] [I1]
[V2] = [Z21 Z22] [I2]

Z11 = V1/I1 | I2=0 (Driving-point)
Z12 = V1/I2 | I1=0 (Transfer)

Y-Parameters (Short Circuit)

[I1] = [Y11 Y12] [V1]
[I2] = [Y21 Y22] [V2]

Y11 = I1/V1 | V2=0 (Admittance)

Mod 4: Two-Port (ABCD & h)

ABCD Parameters (Transmission)

[V1] = [A B] [ V2]
[I1] = [C D] [-I2]

A = V1/V2 | I2=0
B = V1/-I2 | V2=0

h-Parameters (Hybrid)

[V1] = [h11 h12] [I1]
[I2] = [h21 h22] [V2]

Exam Shortcuts

  • Reciprocity Condition:
    Z12 = Z21, Y12 = Y21, AD-BC = 1
  • Symmetry Condition:
    Z11 = Z22, Y11 = Y22, A = D
  • In mesh analysis, if a current source is shared between two meshes, form a Supermesh.
  • In AC Transients, replace Inductor with sL and Capacitor with 1/sC in Laplace domain.
